# 🧩 Sudomind: Gamified AI-Powered Sudoku SaaS

**Sudomind** — это не просто классическое судоку, а современная платформа для тренировки мозга с элементами социальной соревновательности, ИИ-коучингом и уникальными игровыми мирами.

## 🚀 Уникальные фишки (Unique Value Proposition)
* **AI Coach (Sana):** Встроенный помощник на базе Gemini 2.5 Pro, который не просто дает подсказки, а объясняет стратегии решения, помогая пользователю расти.
* **Игровые ниши (Modes):**
    * **Warm-up:** Короткие сессии (180 сек) для "запуска" мозга утром.
    * **Academy Mode:** Интерактивное обучение техникам судоку через ИИ.
    * **Zen Mode:** Игра без таймеров и жизней для медитации.
    * **nFactorial Mode (Dev Mode):** Тематический режим для разработчиков: ошибки — это "Bugs", очки — "Experience".
* **Daily Challenge:** Глобальное ежедневное испытание с единым для всех бордом и общим рейтингом.
* **Социальный слой:** Лидерборды с фильтрацией по городам (например, "Топ игроков Алматы").

## 🎨 Визуальные миры (Themes)
Платформа предлагает 4 глубоко проработанные темы с уникальной анимацией и звуковым сопровождением:
1.  **Clouds** (Светлая и легкая)
2.  **Kawaii** (Пастельные тона и милота)
3.  **nFactorial** (Киберпанк/Консольный стиль)
4.  **Dark Night** (Глубокий темный режим для ночных сессий)

## 💻 Стек технологий
* **Frontend:** Next.js 14 (App Router), TypeScript, Tailwind CSS, Framer Motion.
* **Backend & DB:** Supabase (Auth, PostgreSQL, Realtime).
* **AI Integration:** Google Gemini API (RAG-подход для анализа доски).
* **Monetization:** Готовая структура под Pro-подписку и кастомные скины (Stripe Integration ready).
